## 2. 핵심 정책

- 전화번호 인증 후 비밀번호 재설정.
- 인증번호는 6자리, 유효시간 3분.
- 재전송 쿨다운 60초.

## 4. 인터랙션

### 4.1 인증번호 발송
- 전화번호 입력 완료 시 "인증번호 발송" 버튼 활성화.
- 발송 성공 시 타이머 시작(03:00).

### 4.2 재전송
- 재전송 버튼은 60초 쿨다운 이후 활성화.

### 4.3 비밀번호 재설정
- 인증번호 6자리 입력 완료 시 새 비밀번호 입력 활성화.
- 비밀번호 규칙 미충족/불일치 시 진행 불가.
- "재설정 완료" 성공 시 A00 로그인 섹션으로 이동.
