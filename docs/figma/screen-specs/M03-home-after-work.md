## 4. 자동 퇴근 표시

AUTO_8H 정책으로 자동 퇴근된 경우:

```
┌─────────────────────────────────────┐
│  오늘의 근무                         │
│  ───────────────────────────────── │
│  출근     09:00                     │
│  퇴근     17:00 (자동)              │  ← 자동 퇴근 표시
│  ───────────────────────────────── │
│  근무시간  8시간 00분                 │
└─────────────────────────────────────┘

"(자동)" 라벨: Slate 400, 12px
```

## 5. 날짜 변경 처리

```
근무일 사이클: 04:00 ~ 익일 03:59

04:00 이후:
  - 화면 자동 리셋
  - commuteStatus → WORK_OFF
  - M01 (출근 전) 화면으로 전환
```
