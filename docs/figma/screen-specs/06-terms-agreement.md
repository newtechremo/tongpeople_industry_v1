## 5. 인터랙션

### 5.1 전체 동의 토글

```
체크 시:
  → 모든 개별 약관 체크

해제 시:
  → 모든 개별 약관 해제
```

### 5.2 개별 약관 토글

```
체크/해제 시:
  → 해당 약관만 토글
  → 모든 약관 체크되면 전체 동의 자동 체크
  → 하나라도 해제되면 전체 동의 해제
```

### 5.3 약관 상세 보기

```
화살표(>) 터치 시:
  → 전체 화면 모달로 약관 내용 표시
  → 스크롤 가능
  → 상단 닫기 버튼
```

### 5.4 다음 버튼

```
1. 모든 필수 약관 동의 확인
2. 동의 데이터 저장
3. A07 전자서명 화면으로 이동
```
