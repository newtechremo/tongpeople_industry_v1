## 4. 인터랙션

### 4.1 로그인 버튼
- 전화번호/비밀번호로 로그인 처리.
- 소속 있음: M01~M03 홈, 소속 없음: P04 참여 회사 목록.
 - 실패 메시지: "전화번호 또는 비밀번호가 올바르지 않습니다."

### 4.2 가입 버튼
- A01 회사코드 입력으로 이동 (signup mode).

### 4.3 비밀번호 재설정
- "비밀번호를 잊으셨나요?" 클릭 시 L02로 이동.

### 4.4 관리자 로그인 체크 실패
- 실패 메시지: "관리자 권한이 없습니다."
