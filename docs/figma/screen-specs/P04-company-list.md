# P04 참여 회사 목록 (Company List)

> **화면 ID**: P04
> **파일명**: `P04-company-list.md`
> **상태**: Phase 1 MVP (추가 정의)

---

## 1. 기본 정보

| 항목 | 내용 |
|------|------|
| **화면명** | 참여 회사 목록 |
| **영문명** | Company List |
| **접근 조건** | A00 로그인 성공 (소속 없음 또는 개인 계정) |
| **이전 화면** | A00 로그인/가입 선택 |
| **다음 화면** | A01 회사코드 입력 또는 M01 홈 |

---

## 2. 핵심 정책

- 참여한 회사가 없으면 빈 상태 안내.
- 회사코드를 입력해 소속 추가 가능.

---

## 3. 레이아웃 구조

```
┌─────────────────────────────────────┐
│  ←       참여 회사 목록              │
├─────────────────────────────────────┤
│                                     │
│  회사코드가 있으신가요?              │
│  ┌─────────────────────────────┐   │
│  │  회사코드 입력               │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │         소속 추가            │   │
│  └─────────────────────────────┘   │
│                                     │
│  참여한 회사 (0)                     │
│  ┌─────────────────────────────┐   │
│  │  참여한 회사가 없습니다        │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

---

## 4. 인터랙션

- "소속 추가" 클릭 시 A01 회사코드 입력으로 이동.
- 참여 회사 선택 시 해당 회사 컨텍스트로 이동 (M01 홈).
