## 3. UI 컴포넌트

### 3.1 헤더

```
┌─────────────────────────────────────┐
│  [🏠 산업현장통]                [🔔] │
└─────────────────────────────────────┘

높이: 56px
로고: 왼쪽 16px
알림 아이콘: Bell, 오른쪽 16px
알림 뱃지: 미읽음 있으면 빨간 점
```

### 3.2 인사 영역

| 요소 | 스펙 |
|------|------|
| 인사말 | "안녕하세요, {이름}님" |
| 폰트 | H3 (20px, SemiBold) |
| 서브 | "오늘도 안전한 하루 되세요!" |
| 서브 폰트 | Body L (16px), Slate 500 |

### 3.3 메인 카드

```
┌─────────────────────────────────────┐
│                                     │
│       2026년 1월 13일 (월)          │  ← 오늘 날짜
│                                     │
│          ┌─────────┐               │
│          │ 출근하기 │               │  ← 120x120 원형 버튼
│          └─────────┘               │
│                                     │
│        출근 전입니다                 │  ← 상태 텍스트
│                                     │
└─────────────────────────────────────┘

카드: 흰 배경, 그림자, rounded-2xl
날짜: 16px, SemiBold, Slate 600
버튼: 원형, 120x120, Blue Gradient (from-blue-500 to-blue-600)
상태: 16px, Medium, Slate 500
```

### 3.4 출근하기 버튼 (원형)

```
       ┌─────────────┐
       │             │
       │   출근하기   │
```

## 5. 인터랙션

### 5.1 출근하기 버튼

```
1. 터치 시 scale 애니메이션 (0.95)
2. 확인 팝업: "출근하시겠습니까?"
3. [확인] 시:
   - API 호출: POST /commute-in
   - 성공: M02 (근무 중) 화면으로 전환
   - 토스트: "출근이 완료되었습니다"
4. [취소] 시: 팝업 닫기
```
