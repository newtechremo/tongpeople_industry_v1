## 3. UI 컴포넌트

### 3.1 헤더

| 요소 | 스펙 |
|------|------|
| 높이 | 56px |
| 뒤로가기 | ChevronLeft 24px, Slate 800, 좌측 16px |
| 제목 | 없음 (투명 헤더) |

### 3.2 회사 정보

| 요소 | 스펙 |
|------|------|
| 회사명 | H2 (24px, Bold), Slate 800, 중앙 |
| 위치 | 헤더 아래 40px |

### 3.3 안내 텍스트

| 요소 | 스펙 |
|------|------|
| 텍스트 | "근무하실 현장을 선택해주세요" |
| 폰트 | Body L (16px), Slate 600 |
| 정렬 | 중앙 |
| 위치 | 회사명 아래 8px |

### 3.4 현장 카드 (Site Card)

| 요소 | 스펙 |
|------|------|
| 높이 | 자동 (콘텐츠 기반) |
| 패딩 | 16px |
| 테두리 | 2px, Gray 200 (선택 시: Orange 500) |
| 모서리 | 12px (rounded-xl) |
| 배경 | White (선택 시: Orange 50) |
| 간격 | 카드 간 12px |

#### 카드 내부 구조

```
┌─────────────────────────────────────┐

현장명: 16px, SemiBold, Slate 800
```

## 4. 상태별 UI 변화

### 4.1 Default (초기)

```
모든 카드: 미선택 상태 (border-gray-200)
버튼: 비활성 (bg-gray-200)
```

### 4.2 Selected (선택됨)

```
선택 카드:
  - border-orange-500
  - bg-orange-50
  - 라디오 filled (orange-500)

다른 카드: 미선택 상태 유지
버튼: 활성 (Orange Gradient)
```

## 5. 인터랙션

### 5.1 카드 선택

```
1. 카드 터치 시 즉시 선택 상태로 변경
2. 이전 선택 해제 (단일 선택)
3. 버튼 활성화
```

### 5.2 다음 버튼

```
1. 선택된 현장 정보 저장
2. A03 전화번호 입력 화면으로 이동
```

### 5.3 뒤로가기

```
1. A01 회사코드 입력으로 복귀
2. 입력했던 회사코드 유지
```
