## 3. UI 컴포넌트

### 3.1 성공 아이콘

```
           ✓

아이콘: CheckCircle (Lucide)
크기: 80x80
색상: Green 500
배경: Green 50 (원형, 120x120)
애니메이션:
  - 스케일 0 → 1 (바운스)
  - 체크 마크 그리기 효과
```

### 3.2 메인 메시지

```
   출근이 완료되었습니다
   (또는 "퇴근이 완료되었습니다")

폰트: H2 (24px, Bold)
색상: Slate 800
정렬: 중앙
```

### 3.3 근로자 정보 카드

```
┌─────────────────────────────────────┐
│                                     │
│            홍길동                    │  ← 이름
│     (주)정이앤지 · 전기기사           │  ← 소속/직책
│                                     │
│  ─────────────────────────────────  │
│                                     │
│     출근    09:00                   │  ← 출근 시간
│     (또는 퇴근 18:00)                │
│                                     │
└─────────────────────────────────────┘

배경: White
테두리: Green 200
모서리: 16px
패딩: 24px

이름: 20px, Bold, Slate 800
소속: 14px, Regular, Slate 500
구분선: Gray 200
시간 라벨: 14px, Regular, Slate 500
시간 값: 18px, Bold, Green 600
```

### 3.4 자동 닫힘 카운트다운

```
   3초 후 자동으로 닫힙니다

폰트: 14px, Regular
색상: Slate 400
애니메이션: 숫자 카운트다운 (3, 2, 1)
```

## 7. 인터랙션

### 7.1 자동 복귀

```
1. 성공 화면 표시
2. 3초 카운트다운 시작
3. 0초 도달: Q01 (QR 스캔) 화면으로 자동 전환
4. 모드 유지 (출근 모드였으면 출근 모드 유지)
```

### 7.2 계속 스캔하기 버튼

```
1. 버튼 터치 시 즉시 Q01로 이동
2. 카운트다운 취소
3. 모드 유지
```

### 7.3 화면 터치 (선택적)

```
- 화면 아무 곳이나 터치 시 즉시 Q01로 이동
- 빠른 연속 스캔 지원
```
