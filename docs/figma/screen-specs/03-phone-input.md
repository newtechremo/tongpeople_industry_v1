### 3.3 전화번호 입력 필드

| 요소 | 스펙 |
|------|------|
| 높이 | 56px |
| 형식 | 자동 하이픈 (010-0000-0000) |
| 키보드 | 숫자 패드 |
| 폰트 | 20px, Bold |
| 글자 간격 | 2px |
| 플레이스홀더 | "010-0000-0000" |
| 최대 길이 | 13자 (하이픈 포함) |

```tsx
// 자동 포맷팅
const formatPhone = (value: string) => {
  const numbers = value.replace(/\D/g, '');
  if (numbers.length <= 3) return numbers;
  if (numbers.length <= 7) return `${numbers.slice(0,3)}-${numbers.slice(3)}`;
  return `${numbers.slice(0,3)}-${numbers.slice(3,7)}-${numbers.slice(7,11)}`;
};
```

## 5. 인터랙션

### 5.1 번호 입력

```
1. 숫자만 입력 허용
2. 자동 하이픈 포맷팅
3. 11자 (하이픈 제외) 도달 시 버튼 활성화
```

### 5.2 인증번호 받기 클릭

```
1. API 호출: POST /send-sms
2. 성공 시: A04 인증번호 입력으로 이동
3. 실패 시: 에러 메시지 표시
```

## 6. API 연동

### Request

```typescript
POST /functions/v1/send-sms

{
  "phoneNumber": "01012345678"
}
```

### Response

```typescript
// 성공
{
  "success": true,
  "expiresIn": 180  // 3분
}

// 실패
{
  "success": false,
  "error": "RATE_LIMITED",
  "message": "잠시 후 다시 시도해주세요"
}
```

---

## 7. 에러 처리

| 에러 | 메시지 |
|------|--------|
| `INVALID_PHONE` | 올바른 전화번호를 입력해주세요 |
| `RATE_LIMITED` | 잠시 후 다시 시도해주세요 (1분 후) |
| `BLOCKED_NUMBER` | 사용할 수 없는 번호입니다 |
