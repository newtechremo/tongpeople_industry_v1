## 6. 상태 전이

### 6.1 화면 상태
```
[초기 상태] → [입력 중] → [제출] → [완료/에러]
     │            │          │
     └── 취소 ────┴── 재시도 ─┘
```

### 6.2 상태별 UI
| 상태 | UI 변화 |
|------|---------|
| 초기 | 입력 필드 Default, 버튼 Disabled |
| 입력 중 | 입력 필드 Focused, 버튼 Enabled (유효성 통과 시) |
| 제출 중 | 버튼 Loading, 입력 Disabled |
| 성공 | Toast 표시, 다음 화면 전환 |
| 에러 | 입력 필드 Error, 에러 메시지 표시 |
