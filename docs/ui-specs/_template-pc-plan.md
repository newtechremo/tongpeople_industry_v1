## 7. 예외 처리

### 7.1 에러 상태
| 상태 | 원인 | 사용자 메시지 | 처리 |
|------|------|--------------|------|
| 로딩 실패 | 네트워크 오류 | "데이터를 불러올 수 없습니다" | 재시도 버튼 |
| 빈 결과 | 데이터 없음 | "등록된 항목이 없습니다" | 등록 안내 |
| 권한 없음 | 접근 제한 | "접근 권한이 없습니다" | 이전 페이지 |
