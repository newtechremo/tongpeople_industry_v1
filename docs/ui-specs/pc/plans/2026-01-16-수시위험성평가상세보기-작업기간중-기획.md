## 1. 화면 개요

### 1.1 목적
작업이 진행 중인 수시 위험성평가(빈도·강도 방식) 문서의 상세 내용을 조회하고, 검토 내용/순회점검사항/아차사고 태그 추가, 추가 위험 요인 등록, 조치 결과 등록 등의 기능을 제공하는 화면.

### 1.2 진입 경로
- 위험성평가 목록 → 작업기간중 상태의 수시 위험성평가 문서 클릭

### 1.3 URL 패턴
```
/risk-assessment/adhoc/{assessmentId}
```

## 5. 사용자 시나리오

### 시나리오 1: 검토 내용 추가

1. 빈도·강도 위험요인 카드의 검토 내용 입력 필드에 텍스트 입력
2. `추가하기` 버튼 클릭
3. 입력한 텍스트가 태그로 추가됨 (`-검토하세요`)
4. 태그의 X 버튼으로 삭제 가능

### 시나리오 2: 순회점검사항 추가

1. 선택 정보 입력 영역의 순회점검사항 입력 필드에 텍스트 입력
2. `추가하기` 버튼 클릭
3. 태그로 추가됨 (`-잘했습니다.`)

### 시나리오 3: 추가 위험 요인 등록

1. 추가 위험 요인 섹션 펼치기
2. `+등록하기` 버튼 클릭
3. 모달에서 사진 등록 (선택적)
4. 내용 입력
5. `선택 완료` 버튼 클릭
6. 카드 목록에 추가됨

### 시나리오 4: 조치 결과 등록

1. 조치 결과 섹션 펼치기
2. `+등록하기` 버튼 클릭
3. 모달에서 작업 공종(소분류) 선택
4. 내용 입력
5. `선택 완료` 버튼 클릭
6. "작성 완료!" 성공 메시지 표시
7. 카드 목록에 추가됨

### 시나리오 5: 등록된 항목 수정/삭제

1. 추가 위험 요인 또는 조치 결과 카드의 `수정` 버튼 클릭
2. 수정 모달에서 내용 변경
3. 저장
4. 또는 `삭제` 버튼으로 삭제

## 6. 비즈니스 로직

### 6.1 태그 입력 규칙

- 입력 후 `추가하기` 버튼으로 태그 생성
- 태그는 `-` 접두사로 표시
- 동일 내용 중복 등록 방지
- X 버튼으로 개별 삭제

### 6.2 사진 등록 규칙

- 최대 1장
- 이미지 파일만 허용 (jpg, png, gif 등)
- 개인정보 동의 필수 (최초 1회)
- 업로드 후 썸네일 표시

### 6.3 접이식 섹션

- 기본 접힘 상태
- 클릭 시 펼침/접힘 토글
- 건수 뱃지 표시
