# Risk Assessment Docs

PC 관리자 위험성평가 문서를 정리한 저장소입니다.
스킬 기반 레퍼런스(`docs/ui-specs`)와 구분해서 현재 구현 기준 문서를 유지합니다.

## 구조
- `docs/risk-assessment/pc/README.md` PC 문서 목차
- `docs/risk-assessment/pc/mvp/` MVP 정의/범위/흐름
- `docs/risk-assessment/pc/screen-specs/` 화면별 스펙 (구현 기준)
- `docs/risk-assessment/figma/` 원본 이미지/동영상 레퍼런스
