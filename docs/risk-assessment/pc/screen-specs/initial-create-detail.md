# 최초/정기 위험성평가 만들기 (PC) - 상세 동작

## 1. 결재라인 모달
- 목록: 공용/위험성평가 태그 라인만 노출
- 선택 시 기본 정보 카드의 결재라인 요약/미리보기 업데이트
- 새 결재라인 생성 → 설정 페이지로 이동

## 2. 대분류/소분류 선택
- 대분류 선택 시 소분류 리스트 활성화
- 소분류는 다중 선택 가능
- 사용자 입력 소분류는 별도 플래그로 구분

## 3. 위험요인 선택 모달
- 검색 모드: 기존 위험요인 목록에서 선택
- 직접 입력 모드: 사용자 입력 위험요인/개선대책
- 선택 시 해당 소분류에 누적 추가
- 작업기간은 기본 정보의 기간으로 초기값 설정

## 4. 제출 후 이동
- 성공: 상세 보기로 이동
- 실패: 경고 메시지 표시
